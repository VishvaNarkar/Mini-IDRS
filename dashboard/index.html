About Mini IDRS

Mini IDRS (Intrusion Detection & Response System) is an educational lab project demonstrating automated intrusion response in a VMware-only virtual lab environment.

It monitors network traffic, extracts features in real-time, identifies anomalous patterns using modular detection plugins, and pushes drop actions simultaneously onto the Linux Firewall VM (FORWARD chain) and the Target host itself (INPUT chain).
